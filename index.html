Погружение в аналитику данных:

Учебный опыт:

  • Проведение исследовательского анализа данных (оценка качества данных, предобработка, очистка, категоризация данных, поиск закономерностей и взаимосвязей в данных, составление графиков и диаграмм)
  • Проведение статистического анализа данных (оценка разброса данных, выбор оптимальных метрик для описания данных)
  • Анализ бизнес-показателей (построение маркетинговых и продуктовых воронок бизнеса, определение пользовательских метрик)
  • Работа с большими наборами данных и использование различных статистических, математических и методов машинного обучения, чтобы улучшить понимание данных

Использование технологий:

  • Jupyter Notebook и Google Colab - инструменты для разработки и представления проектов;
  • Библиотеки Python: pandas, numpy, matplotlib, seaborn, plotly;
  • Google Sheets, MS Excel и другие электронные таблицы (функции, сводные таблицы, ВПР);
  • Работа с базами данных с помощью SQL, извлечение данных из веб-ресурсов (парсинг).
  • Python, SQL
  • Математическая статистика, теория вероятности, базы данных, методы обработки больших данных

  • Готовность и желание развиваться в сфере аналитики;
  • Знание Python, SQL, математической статистики, теории вероятности, баз данных, методов обработки больших данных;
  • Опыт работы с библиотеками: pandas, numpy, matplotlib, seaborn, plotly;
